# 92Code.pl app for microcompany-mangement

## Instalation

```
./vendor/bin/sail up
```

### Snippets

Make factory

```bash
./vendor/bin/sail artisan make:factory UserFactory --model=User

```

Seed fresh

```bash
./vendor/bin/sail artisan migrate:fresh --seed
```

Enter to the shell

```bash
./vendor/bin/sail shell
```
